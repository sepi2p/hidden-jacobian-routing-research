# Third-Party Components

This release contains or interfaces with the following upstream research software and checkpoints. Their original terms and citations continue to apply; this repository does not relicense them.

- BlackboxBench model definitions and CIFAR-10 checkpoints: https://github.com/SCLBD/BlackboxBench
- RobustBench models and loading utilities: https://github.com/RobustBench/robustbench
- Square Attack schedule: https://github.com/max-andr/square-attack
- AutoAttack/APGD/FAB evaluation components: https://github.com/fra31/auto-attack
- Torchvision ImageNet architectures and pretrained weights: https://pytorch.org/vision/stable/models.html

The project-authored experiment, analysis, manifest, and regeneration code is separated from downloaded checkpoints, which are not redistributed in Git.
